# Fake News Detection

##  Project Overview
This project uses Machine Learning and Natural Language Processing (NLP) to classify news articles as **Real** or **Fake**.

## Objective
To detect fake news articles using machine learning and text classification techniques.

## 🛠️ Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- NLTK

## Project File
